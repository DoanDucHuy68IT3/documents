# 📂 Kho Lưu Trữ Tài Liệu Cá Nhân

Chào mừng bạn đến với kho lưu trữ tài liệu của **DoanDucHuy68IT3**. Đây là nơi mình dùng để lưu trữ, quản lý và cập nhật các ghi chú, tài liệu học tập cũng như các dự án cá nhân bằng định dạng Markdown.

---

## 📌 Danh Mục Tài Liệu
- **Lập trình:** Các bài tập, mã nguồn mẫu và hướng dẫn cài đặt môi trường.
- **Lý thuyết:** Ghi chú các môn học công nghệ thông tin.
- **Tiện ích:** Các công cụ, phím tắt hay ho cần ghi nhớ.

---

## 📊 Tiến Độ Hoàn Thành

| Tên tài liệu | Định dạng | Trạng thái |
| :--- | :---: | :---: |
| Hướng dẫn Git cơ bản | `.md` | Đã xong |
| Tài liệu chuyên ngành | `.docx` | Đang cập nhật |
| Bài tập lớn | `.md` | Chưa làm |
